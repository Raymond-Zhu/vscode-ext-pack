# README

## Python Devops Pack

Extension Pack for VSCode that contains useful extensions for Python and Devops development 

Contains

* Docker
* Python
* Ansible
* Better Jinja
* Bracket Pair Colorizer
* One Dark Pro
* Vim
* Material Icon Theme
* GitLens
* autoDocString
* CodeRunner
* Python Test Explorer
* Remote Development
* Peacock
* Settings Sync
